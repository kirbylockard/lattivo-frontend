# Lattivo Frontend

Lattivo is a **habit & progress tracking app** designed to help users build routines, track time, and visualize progress with analytics and charts. This repo contains the **Next.js frontend**, styled with **Tailwind CSS v4** and a custom theme system inspired by organic growth and lattice structures.
